LawGlance Desktop Application - README
==================================================

Thank you for downloading LawGlance!

GETTING STARTED
--------------
First-time users:
1. Create a file named ".env" in the same folder as the LawGlance.exe
2. Add your API keys to this file, for example:
   OPENAI_API_KEY=your_openai_api_key_here

RUNNING THE APPLICATION
----------------------
Option 1: Double-click LawGlance.exe to start the application.
Option 2: Right-click and select "Run as administrator" if needed.

After a few moments, the application will launch in your default web browser.
If it doesn't open automatically, navigate to http://localhost:8501 in your browser.

IMPORTANT NOTES
--------------
- This application requires an internet connection to function properly.
- Some antivirus software may flag the executable. This is a false positive
  due to the packaging method used (PyInstaller).
- Please ensure you have a valid OpenAI API key in your .env file.

TROUBLESHOOTING
--------------
If the application doesn't start:
- Check that your .env file is correctly formatted
- Ensure you have a stable internet connection
- Try running as administrator
- Check the logs in the logs/ folder

KNOWN ISSUES
-----------
- First startup may take 30-60 seconds depending on your system
- Some Windows features may trigger security alerts when first running

CONTACT SUPPORT
--------------
If you encounter any issues, please visit:
https://github.com/lawglance/lawglance/issues
