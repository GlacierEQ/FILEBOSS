DeepSeek-Coder API

Advanced AI Code Assistant

Getting Started

DeepSeek-Coder provides a powerful API for code generation, completion, and chat. Use the following endpoints to interact with the AI model.

Code Completion

POST/api/v1/completion

Generate code completions from a given prompt.

# Example curl request curl -X POST http://localhost/api/v1/completion \ -H "Content-Type: application/json" \ -d '{"prompt": "def fibonacci(n):", "max_tokens": 100}'

Code Insertion

POST/api/v1/insertion

Insert code between a prefix and suffix.

# Example curl request curl -X POST http://localhost/api/v1/insertion \ -H "Content-Type: application/json" \ -d '{"prefix": "def add_numbers(a, b):", "suffix": " return result", "max_tokens": 50}'

Chat with AI

POST/api/v1/chat

Chat with the AI model for interactive coding help.

# Example curl request curl -X POST http://localhost/api/v1/chat \ -H "Content-Type: application/json" \ -d '{"messages": [{"role": "user", "content": "Write a function to calculate factorial"}], "max_tokens": 300}'

Health Check

GET/health

Check if the API is operational.

# Example curl request curl http://localhost/health
